## What Was Wrong
The cdsdashboard.gsp page wasn't extracting `patientId` from the URL query parameter before using it.

## What I Fixed
**File**: `omod/src/main/webapp/pages/cdsdashboard.gsp`

Added parameter extraction code:
```groovy
<%
    // Extract patientId from request parameters
    Integer patientId = null
    String patientIdParam = request.getParameter("patientId")
    if (patientIdParam != null && !patientIdParam.isEmpty()) {
        try {
            patientId = Integer.parseInt(patientIdParam)
        } catch (NumberFormatException e) {
            patientId = null
        }
    }
    if (patientId == null) {
        patientId = 0
    }
%>
```
